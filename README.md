# QA Engineer Portfolio — Rofif Asyrof

A clean, modern, and responsive portfolio website designed for a QA Engineer. Built from scratch without frameworks, emphasizing semantic HTML, accessible design, and smooth scroll interactions.

## 🚀 Features

- **No Frameworks**: Pure HTML5, CSS3, and Vanilla JavaScript.
- **Scroll-Stacking Effect**: Custom `requestAnimationFrame` logic for smooth hero-to-about transitions.
- **Fully Responsive**: Adapts seamlessly from desktop to mobile screens.
- **Accessible (a11y)**: Semantic tags, ARIA attributes, and respects `prefers-reduced-motion`.
- **Contact Form**: Integrated with Formspree for serverless form submissions.

## 🛠️ Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/rofif-asyrof/web-portfolio.git
   ```
2. Open `index.html` in your browser, or use a local server (e.g., Live Server extension in VSCode).

## 🚀 Deployment (Vercel)

This project is optimized for deployment on Vercel:
1. Connect your GitHub repository to Vercel.
2. No build settings or build commands are required.
3. Deploy!
